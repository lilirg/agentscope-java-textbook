### 7.4.1 配置加载机制

```

│          Application启动              │

                │
    ┌───────────▼───────────┐
```

### 7.4.2 Spring Boot集成源码分析

#### 7.4.2.1 自动配置类

**源码位置**：`agentscope-spring/src/main/java/com/alibaba/agentscope/spring/AgentScopeAutoConfiguration.java`

```java
package com.alibaba.agentscope.spring;

import com.alibaba.agentscope.core.agent.Agent;
import com.alibaba.agentscope.core.memory.Memory;
import com.alibaba.agentscope.core.tool.Tool;
import org.springframework.boot.autoconfigure.condition.ConditionalOnMissingBean;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;

import java.util.List;

/**
 * AgentScope自动配置类
 * @since v0.2.0
 */
@Configuration
@ConditionalOnProperty(prefix = "agentscope", name = "enabled", havingValue = "true", matchIfMissing = true)
public class AgentScopeAutoConfiguration {
    
    /**
     * 配置Memory Bean
     */
    @Bean
    @ConditionalOnMissingBean(Memory.class)
    public Memory memory(AgentScopeProperties properties) {
        return new InMemoryMemory(properties.getMemory().getMaxMessages());
    }
    
    /**
     * 配置Tool Bean（聚合）
     */
    @Bean
    @ConditionalOnMissingBean
    public List<Tool> tools(List<Tool> toolList) {
        return toolList;
    }
    
    /**
     * 配置Agent Bean
     */
    @Bean
    @ConditionalOnMissingBean(Agent.class)
    public Agent agent(String name, List<Tool> tools, Memory memory) {
        return new SimpleAgent(name, tools, memory);
    }
}
```

**关键点分析**：
- ✅ **自动配置**：`@Configuration` + `@ConditionalOnProperty`
- ✅ **按需创建**：`@ConditionalOnMissingBean`确保不会覆盖用户自定义Bean
- ✅ **配置绑定**：`AgentScopeProperties`自动绑定`agentscope.*`配置

### 7.4.3 AOP代理与扩展点

#### 7.4.3.1 AOP代理示例

**场景**：Agent工具调用的日志记录

```java
package com.alibaba.agentscope.spring.aop;

import org.aspectj.lang.ProceedingJoinPoint;
import org.aspectj.lang.annotation.Around;
import org.aspectj.lang.annotation.Aspect;
import org.slf4j.Logger;
import org.slf4j.LoggerFactory;
import org.springframework.stereotype.Component;

/**
 * Agent工具调用AOP切面
 * @since v0.2.0
 */
@Aspect
@Component
public class AgentToolCallAspect {
    
    private static final Logger logger = LoggerFactory.getLogger(AgentToolCallAspect.class);
    
    /**
     * 切入点：所有Agent的execute方法
     */
    @Around("execution(* com.alibaba.agentscope.core.agent.Agent+.execute(..))")
    public Object logToolCall(ProceedingJoinPoint joinPoint) throws Throwable {
        String methodName = joinPoint.getSignature().getName();
        Object[] args = joinPoint.getArgs();
        
        logger.info("开始执行Tool: {}，参数: {}", methodName, args);
        
        long startTime = System.currentTimeMillis();
        Object result = joinPoint.proceed();
        long endTime = System.currentTimeMillis();
        
        logger.info("Tool执行完成: {}，耗时: {}ms", methodName, (endTime - startTime));
        
        return result;
    }
}
```

**关键点分析**：
- ✅ **切入点表达式**：`execution(* com.alibaba.agentscope.core.agent.Agent+.execute(..))`
- ✅ **性能监控**：记录方法执行耗时
- ✅ **日志记录**：记录方法调用与参数

#### 7.4.3.2 扩展点：自定义Tool

**场景**：注册自定义Tool到Agent

```java
package com.alibaba.agentscope.example.tool;

import com.alibaba.agentscope.core.response.Response;
import com.alibaba.agentscope.core.tool.Tool;

import java.util.HashMap;
import java.util.Map;

/**
 * 自定义WeatherTool
 * @since v0.2.0
 */
public class WeatherTool implements Tool {
    
    @Override
    public String getName() {
        return "weather";
    }
    
    @Override
    public String getDescription() {
        return "查询指定城市的天气";
    }
    
    @Override
    public Map<String, Object> getParameters() {
        Map<String, Object> params = new HashMap<>();
        params.put("city", Map.of("type", "string", "description", "城市名称"));
        return params;
    }
    
    @Override
    public Response execute(Map<String, Object> args) {
        String city = (String) args.get("city");
        // TODO: 调用天气API
        return new Response("城市 " + city + " 的天气是晴朗，15-25℃");
    }
}
```

**Spring Boot注册**：
```java
package com.alibaba.agentscope.example.config;

import com.alibaba.agentscope.core.tool.Tool;
import com.alibaba.agentscope.example.tool.WeatherTool;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;

@Configuration
public class ToolConfiguration {
    
    @Bean
    public Tool weatherTool() {
        return new WeatherTool();
    }
}
```
