## 7.7.1 本章核心知识回顾

### 7.7.1.1 Agent模块源码分析

| 关键点 | 说明 |
|--------|------|

| **执行逻辑** | 调度Tool、记录Memory、生成Response |

### 7.7.1.2 Tool模块源码分析

| 关键点 | 说明 |
|--------|------|
| **接口定义** | `Tool.java`（getName(), execute(), executeAsync()） |
| **回调接口** | `ToolCallback.java`（onSuccess(), onError()） |
| **参数规范** | JSON Schema格式 |

### 7.7.1.3 Spring Boot集成

| 关键点 | 说明 |
|--------|------|
| **自动配置** | `AgentScopeAutoConfiguration.java` |
| **配置绑定** | `@ConfigurationProperties(prefix = "agentscope")` |
| **Bean注册** | `@ConditionalOnMissingBean` + `BeanFactoryPostProcessor` |
