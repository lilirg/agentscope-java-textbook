### 7.7.1 本章核心知识回顾

#### 7.7.1.1 AgentScope-Java架构

| 关键点 | 说明 |
|--------|------|
| **核心类** | `Agent`（抽象类） |
| **工具接口** | `Tool`（接口） |
| **记忆接口** | `Memory`（接口） |
| **响应类** | `Response`（类） |

**架构图**：
```
应用层（Custom Agent）
     ↓ 继承/实现
框架层（AgentScope-Java）
├── Agent（核心类）
├── Tool（工具接口）
├── Memory（记忆接口）
└── Response（响应类）
     ↓ 调用
服务层（LLM API、工具调用、知识库）
```
