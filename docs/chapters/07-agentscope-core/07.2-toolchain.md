## 🚨 源码分析规范（必须遵守）

### 版本管理

**禁止直接修改源码**：
```bash
# ❌ 错误示例：直接修改源码
cd agentscope-java/
vim src/main/java/Agent.java
```

**✅ 正确示例：创建分支并pull request**
```bash
# 正确做法：创建分支
git clone https://github.com/modelscope/agentscope-java.git
cd agentscope-java/
git checkout -b feature/custom-tool-pool

# 修改源码后，提交PR（Pull Request）
git add .
git commit -m "Add custom tool pool support"
git push origin feature/custom-tool-pool
```

### 源码目录结构

```
agentscope-java/
├── agentscope-core/                  # 核心模块（Agent、Tool、Memory）
│   ├── src/main/java/com/alibaba/agentscope/
│   │   ├── agent/                   # Agent相关类
│   │   │   ├── Agent.java          # Agent接口
│   │   │   └── SimpleAgent.java    # Agent实现
│   │   ├── tool/                   # Tool相关类
│   │   │   ├── Tool.java           # Tool接口
│   │   │   └── ToolCallback.java   # Tool回调接口
│   │   ├── memory/                 # Memory相关类
│   │   │   ├── Memory.java         # Memory接口
│   │   │   └── InMemoryMemory.java # 内存实现
│   │   └── core/                   # 核心调度
│   │       └── AgentExecutor.java  # Agent执行器
│   └── pom.xml                      # 核心模块依赖
│
├── agentscope-spring/                # Spring Boot集成模块
│   ├── src/main/java/com/alibaba/agentscope/spring/
│   │   ├── AgentScopeAutoConfiguration.java  # 自动配置
│   │   └── AgentScopeBeanFactoryPostProcessor.java  # Bean注册
│   └── pom.xml                      # Spring模块依赖
│
├── agentscope-examples/              # 示例项目
│   └── src/main/java/com/alibaba/agentscope/examples/
│       ├── SimpleAgentExample.java
│       └── MultiToolAgentExample.java
│
└── pom.xml                           # 父POM
```

## 源码编译与调试

### 克隆源码

```bash
# 方法1：克隆主干（最新代码）
git clone https://github.com/modelscope/agentscope-java.git
cd agentscope-java/

# 方法2：克隆特定版本
git clone https://github.com/modelscope/agentscope-java.git
cd agentscope-java/
git checkout v0.2.0  # 切换到v0.2.0版本
```

### 编译源码

```bash
# 编译核心模块
cd agentscope-core/
mvn clean package -DskipTests

# 编译Spring集成模块
cd agentscope-spring/
mvn clean package -DskipTests

# 编译所有模块
cd agentscope-java/
mvn clean install -DskipTests
```

### 导入IDE

```bash
# IntelliJ IDEA
# File → Open → 选择 agentscope-java/ 目录
# 选择 "Open as Project"
```
