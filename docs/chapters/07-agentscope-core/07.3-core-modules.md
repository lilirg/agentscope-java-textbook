### 7.3.1.1 Agent接口定义

**源码位置**：`agentscope-core/src/main/java/com/alibaba/agentscope/agent/Agent.java`

```java
package com.alibaba.agentscope.core.agent;

import com.alibaba.agentscope.core.response.Response;

/**
 * Agent核心接口
 * @since v0.1.0
 */
public interface Agent {
    
    /**
     * Agent名称
     * @return Agent名称
     */
    String getName();
    
    /**
     * 运行Agent，返回响应
     * @param input 输入内容
     * @return Agent响应
     */
    Response run(String input);
    
    /**
     * 初始化Agent（可选）
     */
    default void initialize() {
        // 默认实现：空
    }
    
    /**
     * 销毁Agent（可选）
     */
    default void destroy() {
        // 默认实现：空
    }
}
```

**关键点分析**：
- ✅ **接口设计**：简单、清晰，仅定义`getName()`与`run()`方法
- ✅ **扩展性**：`initialize()`与`destroy()`提供可选生命周期钩子
- ✅ **返回值**：`Response`封装了Agent的输出（文本/图片/语音等）

### 7.3.1.2 Agent实现类

**源码位置**：`agentscope-core/src/main/java/com/alibaba/agentscope/agent/SimpleAgent.java`

```java
package com.alibaba.agentscope.core.agent;

import com.alibaba.agentscope.core.memory.Memory;
import com.alibaba.agentscope.core.tool.Tool;
import com.alibaba.agentscope.core.response.Response;

import java.util.List;

/**
 * 简单Agent实现
 * @since v0.1.0
 */
public class SimpleAgent implements Agent {
    
    private final String name;
    private final List<Tool> tools;
    private final Memory memory;
    
    public SimpleAgent(String name, List<Tool> tools, Memory memory) {
        this.name = name;
        this.tools = tools;
        this.memory = memory;
    }
    
    @Override
    public String getName() {
        return name;
    }
    
    @Override
    public Response run(String input) {
        // 使用AgentScope框架执行逻辑
        // 1. 从Memory获取历史
        List<Memory.Message> history = memory.getRecent();
        
        // 2. 调用Tools
        String toolResults = tools.stream()
            .map(tool -> tool.getName() + ": " + tool.execute(new HashMap<>()))
            .collect(Collectors.joining("\n"));
        
        // 3. 生成Response（使用LLM API）
        String context = buildContext(input, history, toolResults);
        OpenAIClient openAIClient = new OpenAIClient(apiKey);
        ChatCompletionOptions options = new ChatCompletionOptions(Arrays.asList(
            new ChatMessage(ChatRole.USER, context)
        ));
        ChatCompletions response = openAIClient.getChatCompletions("gpt-3.5-turbo", options);
        String responseContent = response.getChoices().get(0).getMessage().getContent();
        
        return new Response(responseContent, "success");
    }
    
    private String buildContext(String input, List<Memory.Message> history, String toolResults) {
        StringBuilder context = new StringBuilder();
        context.append("【历史对话】\n");
        history.forEach(msg -> context.append(msg.role).append(": ").append(msg.content).append("\n"));
        context.append("\n【工具结果】\n").append(toolResults).append("\n");
        context.append("\n【当前问题】\n").append(input).append("\n");
        return context.toString();
    }
    
    // ... 其他代码
}
```

**关键点分析**：
- ✅ **依赖注入**：通过构造函数注入`tools`与`memory`
- ✅ **职责划分**：Agent负责调度，Tool负责执行，Memory负责记录
- ⚠️ **待实现**：`run()`方法的执行逻辑需要参考框架源码

## 7.3.2 Tool模块源码分析

### 7.3.2.1 Tool接口定义

**源码位置**：`agentscope-core/src/main/java/com/alibaba/agentscope/tool/Tool.java`

```java
package com.alibaba.agentscope.core.tool;

import com.alibaba.agentscope.core.response.Response;

import java.util.Map;

/**
 * Tool核心接口
 * @since v0.1.0
 */
public interface Tool {
    
    /**
     * Tool名称
     * @return Tool名称
     */
    String getName();
    
    /**
     * Tool描述
     * @return Tool描述
     */
    String getDescription();
    
    /**
     * Tool参数 schema（JSON Schema格式）
     * @return 参数Schema
     */
    Map<String, Object> getParameters();
    
    /**
     * 执行Tool
     * @param args Tool参数
     * @return Tool执行结果
     */
    Response execute(Map<String, Object> args);
    
    /**
     * Tool回调接口（异步执行）
     * @param callback 回调接口
     * @param args Tool参数
     */
    void executeAsync(ToolCallback callback, Map<String, Object> args);
}
```

**关键点分析**：
- ✅ **同步/异步**：支持同步执行（`execute()`）与异步执行（`executeAsync()`）
- ✅ **参数规范**：使用JSON Schema定义参数，便于前端生成表单
- ✅ **回调机制**：异步执行时使用`ToolCallback`接收结果

```java
import java.util.List;

    /**
     * 加载记忆（持久化）

}
```

**关键点分析**：
- ✅ **基本操作**：`addMessage()`、`getRecentMessages()`、`clear()`
- ✅ **持久化**：`save()`与`load()`支持记忆持久化（JSON/Redis等）

### 7.3.3.2 内存实现类

**源码位置**：`agentscope-core/src/main/java/com/alibaba/agentscope/memory/InMemoryMemory.java`

```java
package com.alibaba.agentscope.core.memory;

import java.util.LinkedList;
import java.util.List;

/**
 * 内存实现（LinkedList + 固定窗口）
 * @since v0.1.0
 */
public class InMemoryMemory implements Memory {
    
    private final LinkedList<Message> messages = new LinkedList<>();
    private final int maxMessages;
    
    public InMemoryMemory(int maxMessages) {
        this.maxMessages = maxMessages;
    }
    
    @Override
    public void addMessage(String role, String content) {
        messages.addLast(new Message(role, content));
        if (messages.size() > maxMessages) {
            messages.removeFirst(); // 移除最旧消息
        }
    }
    
    @Override
    public List<Message> getRecentMessages(int limit) {
        return messages.stream()
            .skip(Math.max(0, messages.size() - limit))
            .toList();
    }
    
    @Override
    public void clear() {
        messages.clear();
    }
    
    @Override
    public void save() {
        // 持久化到JSON文件
        String json = messages.stream()
            .map(msg -> "{\"role\":\"" + msg.role + "\",\"content\":\"" + msg.content.replace("\"", "\\\"") + "\"}")
            .collect(Collectors.joining(","));
        try {
            Files.writeString(Path.of("memory_" + System.currentTimeMillis() + ".json"), "[" + json + "]");
        } catch (IOException e) {
            logger.error("Failed to save memory", e);
        }
    }
    
    @Override
    public void load() {
        // 从JSON文件加载
        Path path = Path.of("memory_*.json");
        try (DirectoryStream<Path> stream = Files.newDirectoryStream(Paths.get("."), "memory_*.json")) {
            for (Path file : stream) {
                String json = Files.readString(file);
                // 解析JSON并恢复messages
                // 这里省略具体解析逻辑
            }
        } catch (IOException e) {
            logger.error("Failed to load memory", e);
        }
    }
    
    // 内部类：消息定义
    public static class Message {
        private final String role;
        private final String content;
        
        public Message(String role, String content) {
            this.role = role;
            this.content = content;
        }
        
        public String getRole() { return role; }
        public String getContent() { return content; }
    }
}
```

**关键点分析**：
- ✅ **数据结构**：`LinkedList`支持O(1)添加/删除

## 7.3.4 核心模块依赖关系

```
┌──────────────────────────────────────┐
│           Agent（AgentScope-Java）    │
└───────────────┬──────────────────────┘
                │
    ┌───────────┴───────────┐
    │                       │
┌───▼───┐           ┌─────▼──────┐
│ Tool  │           │  Memory    │
│       │           │            │
│ - getName()         │ - addMessage() │
│ - execute()         │ - getRecent()  │
│ - executeAsync()    │ - clear()        │
└───────┘           └──────────────┘
    │                       │
    └───────────┬───────────┘
                │
    ┌───────────▼───────────┐
    │   AgentExecutor       │
    │   （Agent执行器）     │
    └───────────────────────┘
```

**依赖关系说明**：
- ✅ **Agent依赖Tool**：Agent通过Tool执行具体任务
- ✅ **Agent依赖Memory**：Agent通过Memory记录对话历史
- ✅ **Tool与Memory独立**：Tool与Memory无直接依赖，职责清晰
