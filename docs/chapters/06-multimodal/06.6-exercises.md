### 6.6.2 代码改错（每题15分，共30分）

#### 1. 图片预处理改错

```java
// ❌ 错误代码（ImagePreprocessor.java）
public class ImagePreprocessor {
    private static Mat sharedMat = new Mat(); // ❌ 全局共享

    public static float[] preprocess(byte[] imageData) {
        Mat mat = decodeImage(imageData);
        Mat resized = resizeImage(sharedMat, 224, 224); // ❌ 使用共享Mat
        return normalizeImage(resized);
    }
}
```

**问题**：  
1. 全局共享Mat导致线程不安全  
2. 使用了错误的Mat对象

**正确代码**：
```java
public class ImagePreprocessor {
    public static float[] preprocess(byte[] imageData) {
        Mat mat = null;
        try {
            mat = decodeImage(imageData);
            Mat resized = resizeImage(mat, 224, 224); // ✅ 使用正确的Mat
            return normalizeImage(resized);
        } finally {
            if (mat != null) mat.release(); // ✅ 确保释放
        }
    }
}
```

**问题**：  
1. 未处理最后一段的边界条件  
2. 可能导致重复处理

**正确代码**：
```python
def segment_audio(audio_path, output_dir):
    audio = AudioSegment.from_file(audio_path)
    segment_length = 30  # 秒
    overlap = 1  # 秒
    
    for i, start in enumerate(range(0, len(audio), (segment_length - overlap) * 1000)):
        end = start + segment_length * 1000
        segment = audio[start:end]
        
        output_path = f"{output_dir}/segment_{i}.wav"
        segment.export(output_path, format="wav")
        
        if end >= len(audio):  # ✅ 处理边界
```

### 6.6.3 小型设计（30分）

#### 设计任务：多模态输入验证器

**需求**：  
设计一个`MultimodalInputValidator`类，验证输入的有效性。

**功能要求**：  
1. 验证图片字节数组非空且格式合法  
2. 验证音频字节数组非空且可解码  
3. 验证文本非空  

**参考答案**：
```java
public class MultimodalInputValidator {
    /**
     * 验证多模态输入
     * @param input 多模态输入
     * @throws IllegalArgumentException 验证失败时抛出
     */
    public static void validate(MultimodalInput input) {
        if (input == null) {
            throw new IllegalArgumentException("MultimodalInput不能为空");
        }
        
        // 验证文本输入
        if (input.hasText()) {
            String text = input.getText().trim();
            if (text.isEmpty()) {
                throw new IllegalArgumentException("文本输入不能为空");
            }
        }
        
        // 验证图片输入
        if (input.hasImage()) {
            byte[] image = input.getImage();
            if (image == null || image.length == 0) {
                throw new IllegalArgumentException("图片数据不能为空");
            }
            validateImage(image);
        }
        
        // 验证音频输入
        if (input.hasAudio()) {
            byte[] audio = input.getAudio();
            if (audio == null || audio.length == 0) {
                throw new IllegalArgumentException("音频数据不能为空");
            }
            // 实际使用时需验证音频格式（如：检查文件头）
            // 这里仅做演示，不进行严格验证
        }
    }
    
    private static void validateImage(byte[] imageData) {
        // 检查文件头（Magic Number）
        if (imageData.length < 4) {
            throw new IllegalArgumentException("图片数据过短");
        }
        
        // JPEG: FF D8 FF
        if (imageData[0] == (byte)0xFF && imageData[1] == (byte)0xD8 && imageData[2] == (byte)0xFF) {
            return;
        }
        // PNG: 89 50 4E 47
        if (imageData[0] == (byte)0x89 && imageData[1] == (byte)0x50 && imageData[2] == (byte)0x4E && imageData[3] == (byte)0x47) {
            return;
        }
        // GIF: 47 49 46 38
        if (imageData[0] == (byte)0x47 && imageData[1] == (byte)0x49 && imageData[2] == (byte)0x46 && imageData[3] == (byte)0x38) {
            return;
        }
        
        throw new IllegalArgumentException("图片格式不支持");
    }
}
```

### 6.6.4 拓展挑战（20分）

#### 挑战1：图片降采样优化

**背景**：  
当前图片预处理固定resize 224×224，但用户上传的图片可能为4K/8K高清，导致性能浪费。

**问题**：  
1. 如何自动检测图片尺寸并决定是否降采样？  
2. 设计一个智能降采样策略（平衡精度与性能）  
3. 如何缓存预处理结果（避免重复处理）？

**参考答案**：

```java
public class ImagePreprocessor {
    private static final int TARGET_WIDTH = 224;
    private static final int TARGET_HEIGHT = 224;
    private static final int LARGE_IMAGE_THRESHOLD = 1920 * 1080; // 1080p像素数
    
    public static float[] preprocess(byte[] imageData) {
        Mat mat = null;
        try {
            mat = decodeImage(imageData);
            
            // 智能降采样
            int originalPixels = mat.cols() * mat.rows();
            if (originalPixels > LARGE_IMAGE_THRESHOLD) {
                System.out.println("图片过大（" + originalPixels + "像素），进行降采样");
                // 降采样策略：保持宽高比，缩小到1080p
                double scale = Math.sqrt(LARGE_IMAGE_THRESHOLD / (double) originalPixels);
                int newWidth = (int) (mat.cols() * scale);
                int newHeight = (int) (mat.rows() * scale);
                mat = resizeImage(mat, newWidth, newHeight);
            }
            
            // 最终resize到CLIP标准尺寸
            Mat resized = resizeImage(mat, TARGET_WIDTH, TARGET_HEIGHT);
            return normalizeImage(resized);
        } finally {
            if (mat != null) mat.release();
        }
    }
    
    // 缓存预处理结果（简化版）
    private static final Map<String, float[]> cache = new ConcurrentHashMap<>();
    
    public static float[] preprocessCached(byte[] imageData) {
        String key = Arrays.hashCode(imageData) + ""; // 简化：使用哈希作为key
        return cache.computeIfAbsent(key, k -> preprocess(imageData));
    }
}
```

```java
    private static final Logger logger = LoggerFactory.getLogger(WhisperClient.class);
    private static final int MAX_RETRIES = 3;
    private static final long INITIAL_DELAY_MS = 1000;
    
    public String transcribe(byte[] audioData) {
        for (int attempt = 1; attempt <= MAX_RETRIES; attempt++) {
            try {
                // 调用Whisper API
                return transcribeInternal(audioData);
            } catch (SocketTimeoutException e) {
                logger.warn("Whisper API超时（第{}次尝试）", attempt, e);
                
                if (attempt < MAX_RETRIES) {
                    // 指数退避
                    long delay = INITIAL_DELAY_MS * (long) Math.pow(2, attempt - 1);

                        Thread.sleep(delay);
```
