### 6.7.1 本章核心知识回顾

#### 6.7.1.1 图片输入处理（CLIP模型）

| 关键点 | 说明 |
|--------|------|
| **预处理** | OpenCV解码 → resize 224×224 → 归一化 |
| **颜色通道** | OpenCV默认BGR → CLIP需要RGB |
| **输出维度** | 512维/1024维向量 |
| **问题** | 图片过大导致性能浪费、内存溢出 |

**数据流图**：
```
原始图片（BGR） → 解码（IMREAD_COLOR）
     ↓
resize 224×224（Imgproc.resize）
     ↓
转换类型（CV_8U → CV_32F）
     ↓
归一化（÷255.0）
     ↓
BGR → RGB重排
     ↓
CLIP输入（[C,H,W]格式）
```

---

#### 6.7.1.2 语音输入处理（Whisper模型）

| 关键点 | 说明 |
|--------|------|
| **预处理** | pydub格式转换 → 音频分段（30秒窗口） |
| **音频要求** | 16kHz单声道WAV格式 |
| **输出** | 文字序列（中文/英文等） |
| **问题** | ffmpeg未安装、音频过长导致超时 |

**数据流图**：
```
原始音频（mp3/wav/flac）
     ↓
pydub格式转换（16kHz单声道WAV）
     ↓
音频分段（30秒窗口，重叠1秒）
     ↓
Whisper模型编码
     ↓
输出文字（中文/英文）
```

---

#### 6.7.1.3 多模态Agent设计

| 关键点 | 说明 |
|--------|------|
| **输入封装** | MultimodalInput（文本/图片/语音） |
| **特征融合** | 投影到统一空间（全连接层） |
| **线程安全** | ThreadLocal + 深拷贝 |
| **资源管理** | OpenCV Mat.release() + 临时文件清理 |

**架构图**：
```
┌──────────────────────┐
│  多模态输入          │
│  - 文本              │
│  - 图片              │
│  - 语音              │
└──────────────────────┘
    ↓
┌──────────────────────┐
│  预处理              │
│  - 文本：无处理      │
│  - 图片：OpenCV+CLIP │
│  - 语音：pydub+Whisper│
└──────────────────────┘
    ↓
┌──────────────────────┐
│  Agent处理           │
│  - 工具调用          │

```
