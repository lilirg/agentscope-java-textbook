## 🚨 安全规范（必须遵守）

### 禁止硬编码API Key

**❌ 错误示例**：
```java
// ❌ 不要这样做！（除非仅用于本地测试）
public class APIConfig {
    private final String apiKey = System.getenv("OPENAI_WHISPER_API_KEY"); // 从环境变量读取
}
```

**✅ 正确示例**：
```java
// ✅ 使用环境变量（开发环境）
@Value("${openai.whisper.api.key}")
private String apiKey;

// 或使用System.getenv()（生产环境）
private final String apiKey = System.getenv("OPENAI_WHISPER_API_KEY");
```

### 配置文件安全

**`.env`文件模板**（**切勿提交到Git**）：
```env
## .env（添加到.gitignore）
OPENAI_WHISPER_API_KEY=your_openai_whisper_api_key_here
OPENAI_API_KEY=your_openai_api_key_here
CHROMA_ENDPOINT=http://localhost:8000
```

**`.gitignore`配置**：
```gitignore
## 忽略包含密钥的文件
.env
.env.local
.env.*.local
```

### 使用dotenv-java加载配置

```java
// 添加依赖
// <dependency>
//     <groupId>io.github.cdimascio</groupId>
//     <artifactId>dotenv-java</artifactId>
//     <version>3.0.0</version>
// </dependency>

public class ConfigLoader {
    public static String load(String key) {
        Dotenv dotenv = Dotenv.configure()
            .directory(".env") // 指定.env文件目录
            .ignoreIfMissing()
            .load();
        return dotenv.get(key);
    }
}
```

## 环境验证清单

完成以下检查后，确认环境配置正确：

| 检查项 | 命令/操作 | 预期结果 |
|-------|---------|---------|
| JDK版本 | `java -version` | ≥17（如：openjdk version "17.0.9"） |
| Maven版本 | `mvn -version` | ≥3.8（如：Apache Maven 3.8.8） |
| Git配置 | `git config --global user.name` | 已设置用户名 |
| Python版本 | `python --version` | ≥3.8（如：Python 3.9.7） |
| OpenAI Whisper API Key | `echo $OPENAI_WHISPER_API_KEY`（Linux/Mac）或 `echo %OPENAI_WHISPER_API_KEY%`（Windows） | 返回非空字符串 |
| 本地Whisper.cpp | `./whisper.cpp/main -m models/ggml-base.en.bin -f samples/jfk.wav` | 返回识别文本 |

## 快速开始脚本（Windows/Linux）

```bash
## setup.sh / setup.bat（根据系统选择）
## 安装JDK 17（使用sdkman）
sdk install java 17-tem

## 安装Maven（Linux）
sudo apt-get install maven

## 安装Python 3.8+（Linux）
sudo apt-get install python3.9

## 安装Whisper依赖
pip install pydub openai

## 启动Ollama（本地LLM）
ollama serve &

## 启动Chroma（向量数据库）
docker run -p 8000:8000 chromadb/chroma:latest

## 初始化项目（Maven）
mvn archetype:generate \
    -DgroupId=com.textbook.chapter6 \
    -DartifactId=multimodal-agent \
    -DarchetypeArtifactId=maven-archetype-quickstart \
    -DinteractiveMode=false
```
