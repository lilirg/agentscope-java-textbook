### 11.7.5 本章总结

| 维度 | 学习目标 | 实现方式 |
|------|---------|---------|
| **理解能力** | Docker容器化部署、K8s集群部署、Prometheus监控、告警系统、日志分析 | 本节11.3 |
| **实践能力** | Dockerfile编写、K8s配置、Prometheus配置、告警配置、日志分析配置 | 本节11.4 |
| **调试能力** | Docker镜像过大、容器无法启动、Prometheus数据采集失败、告警未触发、日志索引失败 | 本节11.5 |
| **设计能力** | 监控告警系统设计、日志分析系统设计 | 本节11.6 |

**核心收获**：  
1. **Docker容器化部署** = 多阶段构建 + 健康检查 + 资源限制  
2. **Kubernetes集群部署** = Deployment + Service + HPA  
3. **Prometheus监控系统** = Actuator指标收集 + Grafana可视化  
4. **告警系统配置** = AlertManager + 钉钉/邮件告警  
5. **日志分析** = ELK（Elasticsearch + Logstash + Kibana）或Loki（Promtail + Loki + Grafana）
