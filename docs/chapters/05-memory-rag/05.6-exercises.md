## 5.6.2 代码改错（每题15分，共30分）

### 1. 短期记忆改错

```java
// ❌ 错误代码（ShortTermMemory.java）
public class ShortTermMemory {
    private List<Message> history = new ArrayList<>();
    private int maxMessages = 20;

    public void add(Message msg) {
        history.add(msg);
    }

    public List<Message> getRecent() {
        return history; // 直接返回引用
    }
}
```

**问题**：  
1. 未实现固定窗口策略  
2. `getRecent()`返回引用，外部可修改内部状态

**正确代码**：
```java
public class ShortTermMemory {
    private final LinkedList<Message> history = new LinkedList<>();
    private final int maxMessages = 20;

    public void add(Message msg) {
        history.addLast(msg);
        while (history.size() > maxMessages) {
            history.removeFirst(); // 移除最旧消息
        }
    }

    public List<Message> getRecent() {
        return new ArrayList<>(history); // 返回复制列表
    }
}
```

## 5.6.4 拓展挑战（20分）

### 挑战1：RAG相似度阈值调优

**背景**：  
当前RAG检索返回3个文档（相似度：0.85、0.72、0.30），但第3个文档与问题无关。

**问题**：  
1. 如何过滤低相似度文档？  
2. 相似度阈值设置为多少合适？  
3. 如何动态调整阈值（不同领域文档）？

**参考答案**：

```java
public List<String> retrieve(String query, int k, double minSimilarity) {
    Embedding queryEmbedding = sentenceTransformer.encode(query);
    QueryResult result = client.query("documents", 
        List.of(queryEmbedding), k);

    // 过滤低相似度文档
    List<String> relevantDocs = new ArrayList<>();
    for (int i = 0; i < result.getDocuments().size(); i++) {
        double similarity = result.getDistances().get(i);
        if (similarity >= minSimilarity) {
            relevantDocs.add(result.getDocuments().get(i));
        }
    }

    return relevantDocs;
}

// 使用示例
List<String> docs = rag.retrieve("Spring Boot优势", 5, 0.7);
```

**阈值建议**：  
| 领域 | 推荐阈值 | 说明 |
|------|---------|------|
| 通用问答 | 0.6-0.7 | 宽松，覆盖更多相关文档 |
| 专业领域 | 0.8-0.9 | 严格，确保高相关性 |
| 法律/医疗 | 0.9+ | 极严格，避免错误引用 |

### 挑战2：短期记忆压缩优化

**背景**：  
当前短期记忆固定窗口20条，但每条消息平均长度500 tokens，总token数仍可能超出128K限制。

**问题**：  
1. 如何计算实时token数量？  
2. 设计一个自动压缩策略（触发条件 + 压缩算法）  
3. 压缩后的摘要如何整合到提示词中？

**参考答案**：

```java
public class CompressedShortTermMemory extends ShortTermMemory {
    private static final int MAX_TOKENS = 100000;
    private final LLM llm; // 调用LLM生成摘要

    public void add(Message msg) {
        super.add(msg);

        // 1. 计算总token数
        if (calculateTotalTokens() > MAX_TOKENS) {
            compress();
        }
    }

    private void compress() {
        List<Message> recent = getRecent();
        
        // 2. 生成摘要
        String prompt = "将以下对话压缩为3句话摘要（保留关键信息）：\n" + 
                       recent.stream().map(Message::toString).collect(Collectors.joining("\n"));
        String summary = llm.generate(prompt);

        // 3. 清空历史，替换为摘要
        clear();
        add(new Message("system", "[对话摘要]" + summary));
    }

    private int calculateTotalTokens() {
        // TODO: 使用tiktoken-java精确计算（实际使用时需集成tiktoken-java库）
        int tokens = 0;
        for (Message msg : history) {
            tokens += msg.getContent().split("\\s+").length * 1.3;
        }
        return tokens;
    }
}
```
