## 5.7.1 本章核心知识回顾

### 5.7.1.1 短期记忆（Short-Term Memory）

| 关键点 | 说明 |
|--------|------|
| **数据结构** | `LinkedList`（双端队列，O(1)添加/删除） |
| **窗口策略** | 固定窗口（默认20条），超出时FIFO移除最旧消息 |
| **问题** | 多轮对话可能超出上下文窗口限制（128K tokens） |
| **优化** | 摘要压缩、降低窗口大小、截断正文 |

**数据流图**：
```
┌──────────────────┐
│  用户输入        │
└────────┬─────────┘
         │
┌────────▼─────────┐
│  ShortTermMemory │
│  - add()         │
│  - getRecent()   │
└────────┬─────────┘
         │
┌────────▼─────────┐
│  构建提示词      │
│  - 对话历史      │
│  - 当前输入      │
└────────┬─────────┘
         │
┌────────▼─────────┐
│  LLM生成响应    │
└──────────────────┘
```

### 5.7.1.2 长期记忆（Long-Term Memory）

| 关键点 | 说明 |
|--------|------|
| **数据格式** | JSON（跨会话持久化） |
| **缓存策略** | Redis（毫秒级访问） + 文件（断电不丢失） |
| **Key设计** | `user:{userId}`（避免冲突） |
| **问题** | Redis宕机导致缓存失效 |

**数据流图**：
```
┌──────────────────┐
│  用户会话启动    │
└────────┬─────────┘
         │
┌────────▼─────────┐
│  Redis查找       │
│  GET user:u123  │
└────────┬─────────┘
    ┌────┴────┐
    │  存在？  │
    └────┬────┘
      yes│   no
      ┌──▼──┐ ┌──▼──┐
      │ 返回 │ │ 文件读取 │
      └──┬──┘ └──┬──┘
         │        │
      ┌──▼──┐  ┌──▼──┐
      │ 缓存 │  │ 写入Redis│
      └──┬──┘  └──┬──┘
         │        │
      ┌──▼────────▼──┐
      │ 返回用户偏好 │
      └──────────────┘
```

### 5.7.1.3 RAG（Retrieval-Augmented Generation）

| 关键点 | 说明 |
|--------|------|
| **核心步骤** | 向量化 → 检索 → 生成 |
| **向量化模型** | Sentence Transformer（all-MiniLM-L6-v2） |
| **向量数据库** | Chroma（本地）/ Pinecone（云端） |
| **相似度算法** | 余弦相似度（Cosine Similarity） |

**数据流图**：
```
┌──────────────────┐
│  用户提问        │
│  "Spring Boot优势？" │
└────────┬─────────┘
         │
┌────────▼─────────┐
│  向量化          │
│  SentenceTransformer │
└────────┬─────────┘
         │ 向量 [0.12, -0.34, ...]
┌────────▼─────────┐
│  Chroma检索      │
│  query("documents", vector, k=3) │
└────────┬─────────┘
         │ 相似文档
┌────────▼─────────┐
│  生成提示词      │
│  【相关知识库】  │
│  [1] ...         │
│  [2] ...         │
└────────┬─────────┘
         │
┌────────▼─────────┐
│  LLM生成答案    │
└──────────────────┘
```
