## 基础填空（20分）

**说明**：每题2分，共10题，满分20分

1. API核心特性是：______、______、______  
2. HTTP协议核心概念是：______、______、______  
3. RESTful API核心概念是：______、______、______  
4. API文档核心概念是：______、______、______  
5. API安全核心概念是：______、______、______、______  
6. API测试核心概念是：______、______、______  
7. 开发环境必需工具是：______、______、______、______  
8. API Key安全规范是：______、______、______  
9. Maven依赖核心框架是：______  
10. 日志接口是：______

**答案**：
1. 接口定义、数据交换、功能调用  
2. 请求、响应、状态码  
3. 资源、HTTP方法、状态码  
4. 接口定义、请求/响应示例、错误码说明  
5. 认证、授权、加密、防护  
6. 单元测试、集成测试、性能测试  
7. JDK ≥17、IDE ≥2023.1、Maven ≥3.8、Git ≥2.30  
8. 禁止硬编码、使用.env、使用dotenv-java  
9. org.springframework.boot:spring-boot-starter-web:3.2.5  
10. SLF4J

## 代码改错（30分）

**说明**：每题15分，共2题，满分30分

### 题目1：API调用失败（15分）

**问题**：以下是某项目的API调用代码，请找出其中的3个问题并修改：

```java
// ❌ 错误：API调用代码
String apiKey = System.getenv("OPENAI_API_KEY");
String response = callAPI("https://api.example.com/data");
```

**改错要求**：
1. 找出3个问题  
2. 修改问题，完善API调用代码  
3. 每个问题修改得当得5分，满分15分

**参考答案**：
```
问题1：API Key硬编码  
修改：使用.env文件存储API Key

问题2：API Key未加密  
修改：加密存储API Key

问题3：API调用未加密  
修改：使用HTTPS加密API调用
```

## 小型设计（30分）

**说明**：每题15分，共2题，满分30分

### 题目1：API接口设计（15分）

**任务**：设计“查询数据”API接口，要求：

1. URL设计合理  
2. HTTP方法正确  
3. 参数说明完整  

**评分标准**：
- URL设计合理（5分）  
- HTTP方法正确（5分）  
- 参数说明完整（5分）  

**参考答案**：
```
API接口设计：
URL：/api/data
方法：GET
参数：
- id：数据ID（可选）
- type：数据类型（可选）
- page：页码（可选）
- size：每页大小（可选）
```

---

### 题目2：API安全设计（15分）

**任务**：设计API安全方案，要求：

1. API Key管理  
2. HTTPS加密  
3. API签名  

**评分标准**：
- API Key管理合理（5分）  
- HTTPS加密合理（5分）  
- API签名合理（5分）  

**参考答案**：
```
API安全方案：
1. API Key管理：使用.env文件存储API Key，加密存储，定期更换
2. HTTPS加密：使用HTTPS加密API调用
3. API签名：对API调用数据签名，防止数据篡改
```
