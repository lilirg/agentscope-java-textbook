## 提示词原理

### 提示词定义

**提示词（Prompt）** 是引导LLM生成期望输出的指令。

```
Prompt = 指令 + 上下文 + 示例
```

**提示词核心特性**：
| 特性 | 说明 |
|------|------|
| **指令** | 明确的任务目标 |
| **上下文** | 相关背景信息 |
| **示例** | 示例输入输出对 |

### 提示词组成部分

| 部分 | 说明 |
|------|------|
| **指令** | 明确的任务目标 |
| **上下文** | 相关背景信息 |
| **示例** | 示例输入输出对 |

### 提示词工作流程

```
用户输入 → 提示词工程 → LLM推理 → 输出
```
