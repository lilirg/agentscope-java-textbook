### 9.7.1 本章核心知识回顾

#### 9.7.1.1 API Key安全

| 关键点 | 实现要点 |
|--------|---------|
| 环境变量 | `System.getenv("API_KEY")` |
| 加密存储 | AES加密 + Base64编码 |
| 密钥轮换 | 30天有效期 + 备用密钥 |

---

#### 9.7.1.2 权限控制

| 关键点 | 实现要点 |
|--------|---------|
| RBAC | 用户→角色→权限 |
| ABAC | 用户属性→权限 |
| 拦截器 | Spring Interceptor统一校验 |

---

#### 9.7.1.3 数据安全

| 关键点 | 实现要点 |
|--------|---------|
| 加密存储 | AES加密（静态数据） |
| TLS传输 | TLSv1.3（动态数据） |
| 敏感字段 | 手机号、地址、身份证号加密 |

---

#### 9.7.1.4 安全审计

| 关键点 | 实现要点 |
|--------|---------|
| 日志记录 | SLF4J + Logback |
| 异常检测 | 单位时间API调用统计 |
| 告警系统 | 钉钉/邮件告警 |
