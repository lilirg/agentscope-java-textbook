### 1.2.6 安全规范

#### ⚠️ API Key安全

| 规范 | 说明 |
|------|------|
| **禁止硬编码** | 不得在代码中直接写API Key |
| **使用.env** | 使用`.env`文件存储API Key |
| **使用dotenv-java** | 使用`dotenv-java`库加载`.env` |
| **使用System.getenv()** | 使用`System.getenv()`读取环境变量 |

**示例**：
```java
// ❌ 错误：硬编码API Key
String apiKey = System.getenv("OPENAI_API_KEY");

// ✅ 正确：使用.env + dotenv-java
Dotenv dotenv = Dotenv.configure().load();
String apiKey = dotenv.get("OPENAI_API_KEY");

// ✅ 正确：使用System.getenv()
String apiKey = System.getenv("OPENAI_API_KEY");
```
