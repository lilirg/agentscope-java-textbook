#### 3. 缓存库

| 库 | Maven坐标 | 说明 |
|----|-----------|------|
| **Caffeine** | `com.github.ben-manes.caffeine:caffeine:3.1.8` | 本地缓存（高性能） |
| **Redis** | `org.springframework.boot:spring-boot-starter-data-redis:3.2.5` | 分布式缓存 |

**Caffeine使用示例**：
```java
import com.github.benmanes.caffeine.cache.Cache;
import com.github.benmanes.caffeine.cache.Caffeine;

public class LocalCache {
    private final Cache<String, String> cache = Caffeine.newBuilder()
        .maximumSize(10_000)
        .expireAfterWrite(10, java.util.concurrent.TimeUnit.MINUTES)
        .build();

    public String get(String key) {
        return cache.getIfPresent(key);
    }

    public void put(String key, String value) {
        cache.put(key, value);
    }
}
```

### 10.2.5 性能优化技巧

#### 1. 缓存策略

| 策略 | 说明 | 适用场景 |
|------|------|---------|
| **本地缓存** | Caffeine缓存（高性能） | 单机部署 |
| **分布式缓存** | Redis缓存（跨节点） | 分布式部署 |
| **页面缓存** | HTTP缓存（减少API调用） | Web应用 |

**缓存示例**：
```java
@Service
public class AgentService {
    private final LocalCache localCache = new LocalCache();
    private final RemoteCache remoteCache = new RemoteCache();

    public String processQuery(String query) {
        // 1. 检查本地缓存
        String cached = localCache.get(query);
        if (cached != null) {
            return cached;
        }

        // 2. 检查分布式缓存
        cached = remoteCache.get(query);
        if (cached != null) {
            localCache.put(query, cached); // 同步到本地缓存
            return cached;
        }

        // 3. 调用Agent处理
        String result = agent.process(query);

        // 4. 缓存结果
        localCache.put(query, result);
        remoteCache.put(query, result);

        return result;
    }
}
```

---

#### 2. 批量处理

| 策略 | 说明 | 适用场景 |
|------|------|---------|
| **批量API调用** | 一次调用多个查询 | 多个查询场景 |
| **批量数据处理** | 一次处理多条数据 | 数据处理场景 |

**批量API调用示例**：
```java
public class BatchAPI {
    private final APIClient apiClient;

    public List<String> processBatchQueries(List<String> queries) {
        // 1. 分组（每组10个查询）
        List<List<String>> groups = Lists.partition(queries, 10);

        // 2. 批量处理
        List<String> results = new ArrayList<>();
        for (List<String> group : groups) {
            results.addAll(apiClient.batchProcess(group));
        }

        return results;
    }
}
```
